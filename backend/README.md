# Backend

## Requirements

- Python 3.11+
- [uv](https://docs.astral.sh/uv/getting-started/installation/) installed

## Installation

- `uv sync`
- Do `.env` file according to `.env.template`
 
## Start in dev mode

- `uv run python -m app.main --reload`

## Run tests

- `uv run pytest <specific test path>`
